A First PHP File & The "echo" Command: Display Text On A Website
----------------------------------------------------------------
- Launch the "XAMPP Control Panel"
    - Start the service "Apache Web Server"
    - Launch the Chrome browser
    - Launch VS Code editor


- What happens behind-the-scenes
    - After a PHP file "index.php" is loaded on the browser

    - The XAMPP service "Apache Web Server"
        - Identifies the file "index.php" as a PHP file by its filename extension
        - Hands over the file to XAMPP's PHP interpreter

    - The PHP interpreter
        - Receives the file & parses through the code base
        - Checks for PHP keywords, functions, etc.
        - Renders the corresponding output
        - Hands over the generated output back to the "Apache Web Server"

    - The XAMPP service "Apache Web Server"
        - Receives the generated output from the PHP interpreter
        - Serve the content to the client (Chrome browser)

    - The Chrome browser (client)
        - Renders the content on its currently active window


- To insert (inline) PHP code w/in a HTML document
    - Embed the PHP code between the pair of tags "<?php" & "?>"
        - Tag "<?php"   = Start of a PHP code block
        - Tag "?>"      = End of a PHP code block

    - BOTH tags are NOT rendered on the HTML document 
        - Rendered content by the PHP code becomes content of the HTML document

        - Evident using Chrome browser's "View page source"

    Syntax:
    <?php <PHP_code> ?>


- The PHP "echo" statement
    - It is a language construct that
        - Outputs an expression passed as an argument  value
        - NOTE: It is not a function

    - It does NOT
        - Add a "newline" or "space" character to the expression
        - Have a return value (i.e. void)

    - Reference (https://www.php.net/manual/en/function.echo.php)

    Syntax:
    echo "<string_expression>";
    echo '<string_expression>';
    echo "$<variable_name>";
    echo <expression>;


- The PHP function "rand()"
    - It generates & returns a random number

    - It can optionally accepts two (2) argument values
        - 1st   = minmum (lowest) integer value
        - 2nd   = maximum (highest) integer value

    - When called w/out passing an argument value
        - Returns an integer value between
            - Integer value of 0
            - Integer returned by the function "getrandmax()"
    
    - Reference (https://www.php.net/manual/en/function.rand.php)

    Syntax:
    rand();
    rand([<min_integer_to_generate>, <max_integer_to_generate>]);


- On the project folder
    - Launch XAMPP
        - Run the service "Web Apache Server"

    - On VS Code
        - Create a new file named "index.php"

    - Inside the file "index.php"
        - Generate a HTML boilerplate using the shortcut "!" + "tab"
            
        - Create an HTML element <h1>
            - Embed PHP code as content using an "echo" statement
                - To render the following string value

                "Hello from PHP!"

        - Create a 2nd HTML element <h1>
            - Pass the following string value as content

                "Hey, this is my first .php file"

        - Create a 3rd HTML element <h1>
            - Embed PHP as content using an "echo" statement
                - To call the function "rand()"
                
                - Pass the following argument values
                    - 1st = 1 (lowest generated number)
                    - 2nd = 6 (highest generated number)

    - Save the changes
    
    - Launch the browser
        - Open the file "index.php"
        - Renders the HTML elements <h1> & the corresponding content
