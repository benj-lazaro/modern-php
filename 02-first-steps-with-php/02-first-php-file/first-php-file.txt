A First PHP File & The "echo" Command: Display Text On A Website
----------------------------------------------------------------
- Launch the XAMPP Control Panel
    - Start the "Apache Web Server" service
    - Launch the Chrome browser
    - Launch VS Code editor


- On the project folder
    - Create a new file named "index.php"

    - Generate a HTML boilerplate code
        - Using VS Code shortcut "!"

    - Create an HTML element <h1> w/ the string "Hey, this is my first .php file"
    - Save the changes
    - Reload the browser


- What happens behind the scenes
    - The Apache web server 
        - Identifies the file as a PHP file by virtue of the file extension ".php"
        - Hands over the file to the PHP interpreter

    - The PHP interpreter
        - Parses through the "index.php" file's code base
            - Checks for PHP keywords, functions, etc.
            - Renders the corresponding output
        
        - Hands it back to the Apache web server

    - The Apache web server
        - Receives the content from the PHP interpreter
        - Servers it to the (client) Chrome browser

    - The Chrome browser
        - Renders the content on its (currently active) window


- To insert PHP code w/in a HTML document
    - Use the following syntax:

        <?php <place_PHP_code_here> ?>

    - The "<?php" informs the Apache web server that
        - This is the start of a PHP code block

    - The "?>" signifies the end of a PHP code block

    - IMPORTANT: BOTH "<?php" & "?>" are NOT rendered in the HTML document
        - When viewed using the Chrome browser's "View page source"

        - The PHP code is embedded as content w/in a HTML document

    
- The "echo" statement
    - It is a PHP language construct that
        - Outputs one (1) or more expression passed as an argument value
        - It is NOT a function
    
    - It does NOT
        - Add a newline or space to the expression
        - Have a return value (i.e. void)

    - Reference (https://www.php.net/manual/en/function.echo.php)

    Syntax:
    echo "<string_expression>";
    echo '<string_expression>';
    echo "$<variable_name>";
    echo <expression>;


- The function "rand()"
    - Generates a random integer

    - Optionally accepts two (2) arguments
        - 1st = minimum integer value to generate
        - 2nd = maximum integer value to generate 

    - If called w/o an argument value
        - Returns an integer between 0 & the number returned by the function "getrandmax()"
    
    - Reference (https://www.php.net/manual/en/function.rand.php)

    Syntax:
    rand();
    rand([<min_integer_to_generate>, <max_integer_to_generate>]);
