Quiz 13: Test Your Knowledge Of Databases
-----------------------------------------
- What does a CRUD application allow a user to do?
    - Answer: Perform Create, Read, Update and Delete operations on data


- Which of the following statements about MySQL and MariaDB is incorrect?
    - Answer: MySQL and MariaDB offer the same functionality & efficiency


- What is a schema in a database?
    - Answer: The structure that defines how data is stored, organized & manipulated in a database


- What is the function of a primary key in a database?
    - Answer: To uniquely identify each record in a database table
