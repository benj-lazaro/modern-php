    - How to

    - MySQL / MariaDB realtional database
        - MySQL was bought by Oracle
        
        - Open source developers forked MySQL into MariaDB
            - Have since been independently developed from MySQL

        - Difference between the two 
            - Are based on their advanced features
            - Features that are generally irrelevant for application development
            - Other than that, these two are completely interchangeable
