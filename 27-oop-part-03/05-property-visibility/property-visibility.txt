Property Visibility: Private And Protected Properties For Secure Data Handling
------------------------------------------------------------------------------
- The visibility keyword "private" 
    - Prevents a Class Property to be accessed directly by its instantiated Object
    - Prevents a Child Class to inherit a Parent Class Property


- The visibility "protected"
    - Prevents a Class Property to be accessed directly by its instantiated Object
    - Allows a Child Class to inherit a Parent Class Property


- NOTE: Visibility keywords "private" & "protected" can be applied a Class Method
