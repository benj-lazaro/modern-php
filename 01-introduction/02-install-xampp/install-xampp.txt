Install XAMPP (Windows & Linux)
-------------------------------
- Download the latest installer from the official website
    - Website (https://www.apachefriends.org/download.html)


- Installation (Windows)
    - Run the executable installer
        - Use the default settings

    - Post-installation tasks
        - Launch the XAMPP Control Panel
        
        - Click the "Start" button of the following services
            - Apache
            - MySQL
        
        - On the "Apache" service
            - Click the "Admin" button
            - Launches the system default browser
            - Loads the XAMPP dashboard


- Installation (Linux)
    - Launch terminal
        - Navigate to the directory where the installer is downloaded

    - Make the installer executable

        sudo chmod +x xammp-linux-x64*


    - Execute the installer

        sudo ./xammp-linux-x64*

    
    - Change the ownership of the directory "/opt/lampp/htdocs"

        sudo chown -R <username>:<username> /opt/lampp/htdocs/


    - Launch the XAMPP Control Panel

        sudo /opt/lampp/manager-linux-x64.run


- Troubleshooting Tips
    - Windows OS: XAMPP services failed to start (e.g. Apache, MySQL)
        - The default ports may have been taken by another application or service

        - On the XAMPP Control Panel
            - Click the "Config" button

            - Select the item "Apache (httpd.conf)"
                - Launches default text editor & loads the .conf file

            - Scroll down until the reaching the directive

                Listen 80

            - Change the port number into something else (e.g. 8080 or 8090)

                Listen 8090

            - Close the default editor
                - Automatically save the changes
        

        - Back on the XAMPP Control Panel
            - Click "Config" button again
                - Select the item "Apache (httpd-ssl.conf)"
                - Launches default text editor & loads the .conf file

            - Scroll down until the reaching the directive

                Listen 443

            - Change the port number into something else (e.g. 1891)

                Listen 1891

            - Close the default editor
                - Automatically save the changes


        - Click the "Start" button of the Apache service
            - The updated port numbers are listed under the "Port(s)" column


        - Click the "Admin" button to launch the XAMPP dashboard on the browser
            - If the XAMPP dashboard fails to load, manuall paste the following URL
                - NOTE: Use the assigned ports number

            - URL (https://localhost:<new_port_number>/dashboard)
