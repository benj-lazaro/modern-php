Introduction
------------
- PHP history
    - Launched in 1997
    - Nowadays, powering 80% of all websites
    - Popular software (i.e. WordPress) is written in it
    - PHP is incredibly a powerful programming language
    - PHP hosting is incredibly cheap


- Moreover
    - PHP has significantly evolved over the years

    - It used to be a pure scripting language for simple scripts
        - Might have a bad reputation due to that

    - It has become an advanced, object-oriented programming language that
        - Can be utlizied for modern applications
