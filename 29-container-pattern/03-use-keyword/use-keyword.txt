The "use" Keyword: Weave External Data Into Closures
----------------------------------------------------
- A declared variable outside of a function
    - Is INACCESSIBLE w/in the function's closure

    - To access the external variable
        - Use the keyword "use()"
            - Pass the external variable as an argument value
            - Place it before the opening curly brace of the function

        - It IMPORTS the external variable w/in the closure


Syntax:

$<external_variable> = <data>;

$<variable_name> = function() use($<external_variable>) {
    ...
};

$<variable_name>();
